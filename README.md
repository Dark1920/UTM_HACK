# ArtisanBF

Plateforme geolocalisee pour les artisans locaux du Burkina Faso.

## Description

ArtisanBF est une plateforme qui connecte les artisans locaux a leurs clients grace a un annuaire geolocalise, avec des outils d'intelligence artificielle pour l'analyse de commentaires et la recherche vocale.
